Second sub-task
    Quick eplanation explaining the format:
        * 4 bytes: magic "LEGOSET"
        * 1 byte: version
        * 2 bytes: set_id length, then set_id bytes (utf-8)
        * 2 bytes: set_name length, then set_name bytes (utf-8)
        * 4 bytes: number of brick records

        * per brick:
            2 bytes: image URL length, then bytes
            2 bytes: brick name length, then bytes
            2 bytes: color length, then bytes
            2 bytes: count

    How to test read:
        1. Start server in one terminal: python serber.py

        2: run command in other terminal:
            curl -s "http://127.0.0.1:5001/api/write/set?id=00-1" -o /tmp/set.bin

        3: Decode using read function by running this command in other terminal:
            python - <<'PY'
            from server import read
            with open("/tmp/set.bin", "rb") as f:
                read(f)
            PY
